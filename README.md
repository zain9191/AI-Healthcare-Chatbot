# AI Healthcare Chatbot (A.U. 2025-2026)

## Description
An AI-powered medical chatbot using Decision Tree Classifiers and Natural Language Processing to diagnose diseases based on symptoms.

## Installation
1. Install dependencies:
   `pip install -r requirements.txt`
2. Train the model (Run once):
   `python model.py`
3. Start the application:
   `python app.py`

## Features
- Symptom Extraction from natural text.
- Disease Prediction.
- Specialist Recommendation.
- Voice Interaction (Text-to-Speech).
